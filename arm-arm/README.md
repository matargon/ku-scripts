# ku-scripts

## Назначение
Запускает `run_on_server.sh` на списке серверов из `servers.txt`, пишет общий лог и
складывает неуспешные IP в `failed_ips.txt`.

## Быстрый старт
1) Создай `servers.txt` с одним IP/хостом на строку.
2) Убедись, что `run_on_server.sh` исполняемый: `chmod +x run_on_server.sh`.
3) Запусти: `./main.sh`.

## Файлы
- `main.sh` — основной скрипт, читает список серверов и запускает обработку.
- `run_on_server.sh` — скрипт, который выполняется на каждом сервере.
- `servers.txt` — список серверов (не хранится в Git).
- `all_servers.log` — общий лог запуска (не хранится в Git).
- `failed_ips.txt` — список серверов с ошибками (не хранится в Git).

## Примечания
- Локальные IP пропускаются автоматически.
- Для Xubuntu используется `ip` вместо `ifconfig`.
