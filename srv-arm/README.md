## Назначение
Скрипт для автозагрузки данных на ноутбуки: забирает CSV с сервера, ищет строки
для текущего IP ноутбука, выполняет `rsync` с сервера на ноутбук и отправляет
инкрементальный лог обратно на сервер.

## Формат CSV
Одна строка:
```
ip,src,dst
```
где:
- `ip` — IP ноутбука (совпадает с текущим IP машины)
- `src` — путь на сервере
- `dst` — путь на ноутбуке

## Настройка
Открой `main.sh` и заполни:
- `SERVER_USER` — пользователь на сервере
- `SERVER_HOST` — адрес сервера
- `SERVER_CSV_PATH` — полный путь к CSV на сервере
- `SERVER_LOG_ROOT` — корневая папка для логов на сервере
- `SSH_KEY` — путь к SSH ключу (если не нужен — оставь пустым)

## Запуск
```
chmod +x main.sh
./main.sh
```

## Логи
Для каждого запуска создается отдельный файл на сервере:
```
<SERVER_LOG_ROOT>/<IP>/<IP>_YYYY-MM-DD_HH-MM-SS.log
```
